About Dark Tower

Dark Tower is best experienced using headphones.

Programming and implementation: Pete Haughie

Writing, audio and interface: Astrid Bin

Concept by both.

The technology used was the JavaScript BUZZ library by Jay Salvat to handle audio, jQuery for ease of deployment, AUDACITY for audio production and TWINE for structuring the text.
